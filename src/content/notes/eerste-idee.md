Ik wil een web van ideeën.

Dit linkt naar [[tweede-idee]] omdat het erop voortbouwt.

Een code-snippet:

```ts
type Note = { slug: string; title: string };
```

En een afbeelding zou hier kunnen staan (plaats een afbeelding in `src/assets/` en verwijs ernaar met relatieve pad).

## Waarom een wiki-stijl?

Een traditionele blog is lineair - posts volgen elkaar op in de tijd. Maar ideeën zijn niet lineair. Ze bouwen op elkaar voort, refereren naar elkaar, en vormen een netwerk.

Met wiki-links zoals [[tweede-idee]] kan ik deze verbindingen expliciet maken.

## Voordelen

- **Ontdekking**: Backlinks laten zien waar een idee wordt gebruikt
- **Context**: Links geven extra context zonder de flow te onderbreken
- **Organisch**: Het web van ideeën groeit natuurlijk

Dit is het begin van mijn digitale tuin.
